# Tanuj Dontaraju - Personal Portfolio

**🌐 [Visit the live website here!](https://tanujdontaraju.github.io/Portfolio/)**

My personal portfolio website! I am an AI/ML Engineer studying at the University of Delaware. This sit showcases my coursework, experience, and projects.

## 🚀 Features

*   **Interactive UI**: Smooth scrolling, ripple effects, and dynamic dropdown/accordion sections for coursework and experience.
*   **Custom Animations**: Features a unique animated hexagon background and scroll-triggered animations using the Intersection Observer API.
*   **Responsive Design**: Built with modern CSS techniques to ensure the site looks great on all screen sizes.
*   **Social Connectivity**: Quick access to my LinkedIn, GitHub, Instagram, and direct email via a fixed floating side panel.

## 🛠️ Technologies Used

*   **HTML5**: Semantic structure and native `<details>`/`<summary>` elements for accessible accordions.
*   **CSS3**: Flexbox, CSS Grid, custom keyframe animations, and modern styling (utilizing the *Poppins* font).
*   **Vanilla JavaScript**: DOM manipulation, event handling, and custom animation logic without the need for heavy external frameworks.
